## 2. –°–≤—è–∑—å —Å "–ú–∞—Ç—Ä–∏—Ü–∞–º–∏ –¥–æ–ª–∂–Ω–æ—Å—Ç–Ω—ã—Ö —Ä–æ–ª–µ–π" (–ú–î–†)
