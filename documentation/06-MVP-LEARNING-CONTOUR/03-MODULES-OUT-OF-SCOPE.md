# MODULES-OUT-OF-SCOPE — MVP Learning Contour (v1.1)

## 0. Назначение документа

Данный документ фиксирует перечень функций, КАТЕГОРИЧЕСКИ ЗАПРЕЩЁННЫХ в MVP Learning Contour v1.1.

## 1. Запрещённый функционал

### 1.1 Управление и контроль
- **Автоматические взыскания**: Система не имеет права автоматически аннулировать прогресс без подтверждения HR/Security в Audit Log.
- **KPI-интеграция**: Прямое влияние учебных баллов на финансовые бонусы без экспертной оценки.

### 1.2 Экономика
- **MatrixCoin Marketplace**: Обмен MC на реальные товары или привилегии (временно недоступно в v1.1).
- **GMC (Global MatrixCoin)**: Использование глобальной валюты.

### 1.3 AI-поведение
- **AI-авторитаризм**: Принятие AI самостоятельных решений о допуске сотрудников к работе на основе обучения.
- **Скрытая слежка**: Использование AI для анализа эмоций через камеру или паттерны ввода (только пассивная аналитика логов).

## 2. Запрещённые интерфейсы
- **Скрытые админки**: Интерфейсы, позволяющие руководителям изменять результаты тестов сотрудников напрямую, минуя Event Registry.

## 3. Статус документа
- **Статус**: Active / Canonical (Update v1.1)
- **Контур**: MVP Learning Contour
