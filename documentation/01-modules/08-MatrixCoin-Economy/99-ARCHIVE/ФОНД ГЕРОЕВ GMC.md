I. НАЗНАЧЕНИЕ ФОНДА ГЕРОЕВ
1. Смысл GMC в системе MatrixGin v8

GMC (Golden Matrix Coin) является стратегическим активом участия, а не:

наградой;

бонусом;

улучшенной версией MC;

результатом накопления MC.

GMC фиксирует факт системного участия человека в судьбе компании, а не его операционную активность.

GMC:

не сгорает;

не является регулярным;

не может быть гарантирован;

не может быть получен автоматически.

Любая выдача GMC вне правил настоящего документа считается нарушением канона v8.

II. ПОЧЕМУ ФОНД ГЕРОЕВ ОБЯЗАТЕЛЕН
2. Стратегическая проблема, которую он закрывает

Если GMC выдаются:

только через аукционы;

только через Store;

только как результат накопления MC,

то GMC:

превращается в «дорогой MC»;

теряет смысл стратегического признания;

начинает восприниматься как вознаграждение за активность, а не за вклад.

Фонд Героев — единственный допустимый контур GMC вне рынка.

III. ОПРЕДЕЛЕНИЕ ГЕРОИЗМА (КАНОН v8)
3. Формула героизма

Героизм = осознанный выход за минимально ожидаемую роль
(определённую Матрицей должностной роли),
приведший к системному эффекту или предотвращению системного ущерба.

Ключевые элементы (все обязательны):

осознанность;

выход за роль;

высокий риск / ответственность;

эффект на уровне системы, а не задачи.

IV. ДОПУСТИМЫЕ ТИПЫ ГЕРОИЧЕСКОГО ВКЛАДА

Фонд Героев рассматривает ТОЛЬКО следующие типы вкладов.

TYPE H1 — SYSTEM_SAVING

Смысл:
Предотвращение или купирование системного сбоя, кризиса, потери.

Примеры логики (не кейсов):

предотвращён коллапс процесса;

устранена критическая уязвимость;

сохранён ключевой актив / партнёрство.

Запрещено:

«хорошо справился с кризисом по роли»;

«выполнил антикризисную задачу по должности».

TYPE H2 — SYSTEM_BUILDING

Смысл:
Создание нового системного слоя, изменившего правила игры.

Примеры логики:

запуск нового направления;

создание архитектуры, переживающей автора;

формирование нового устойчивого паттерна.

TYPE H3 — SYSTEM_TRANSFORMATION

Смысл:
Радикальное улучшение существующей системы с долговременным эффектом.

Примеры логики:

слом неэффективного, но устойчивого паттерна;

трансформация культуры или процессов;

переход системы в новое качество.

V. ДИАПАЗОНЫ GMC
5. Диапазоны выдачи (ориентиры, не обязательства)

H1 (System Saving) → 1–3 GMC

H2 (System Building) → 2–5 GMC

H3 (System Transformation) → 3–7 GMC

⚠️ Диапазоны:

не являются гарантией;

не являются шкалой оценки заслуг;

служат ориентиром для комиссии.

VI. ПРОЦЕСС ВЫДВИЖЕНИЯ И РАССМОТРЕНИЯ
6. Источники выдвижения

Кандидат на GMC может быть выдвинут:

руководителем;

коллегиально (группа ≥ 3 человек);

Советом;

через постфактум-разбор (incident / postmortem).

❌ Самовыдвижение запрещено.

7. Комиссия Фонда Героев

Состав (минимальный):

представитель управления;

представитель архитектуры / системы;

независимый участник (не из цепочки подчинения).

Принципы работы:

коллегиальность;

отсутствие кворума = REJECT;

решение фиксируется письменно.

8. Формат решения

Каждое решение комиссии обязано содержать:

тип героического вклада (H1 / H2 / H3);

описание системного эффекта;

аргументацию «почему это выход за роль»;

итог: APPROVED / REJECTED;

объём GMC (если APPROVED).

❌ Запрещены:

«за заслуги»;

«за вклад в целом»;

«за старание».

VII. ПУБЛИЧНАЯ ФИКСАЦИЯ
9. Принцип публичности

Факт выдачи GMC публичен.

Обоснование — краткое и системное, без эмоций.

Обсуждение решения не допускается.

Цель публичности:

формирование ориентиров;

фиксация эталонов героизма;

предотвращение слухов и домыслов.
