- [x] Экономика модуля **не является** финансовой системой
- [x] Экономика модуля **не используется** как KPI, OKR или рычаг давления

## 2. Semantic Integrity (смысловая целостность)

### 2.1 Matrix Coin (MC)

- [x] MC — **операционная, временная, поведенческая единица**
- [x] MC **не является** деньгами, зарплатой, бонусом или их эквивалентом
- [x] MC **не имеет** фиксированного денежного курса
- [x] MC используется только внутри системы MatrixGin
- [x] MC отражает участие и осознанное поведение, а не результат любой ценой

### 2.2 Golden Matrix Coin (GMC)

- [x] GMC — **стратегический статусный актив признания**
- [x] GMC **не является** наградой, призом или бонусом
- [x] GMC **не начисляется автоматически**
- [x] GMC **не конвертируется напрямую** в деньги или MC
- [x] GMC отражает долгосрочное признание вклада в систему

---

## 3. Store & Exchange Rules

### 3.1 Store

- [x] Store реализует **немонетарный обмен**
- [x] Store **не имитирует магазин с ценами**
- [x] Store не формирует у пользователя ощущения «покупки за деньги»
- [x] Объекты Store имеют смысловую, а не финансовую ценность

### 3.2 Auction

- [x] Auction — **событийный шлюз**, а не постоянный механизм
- [x] Auction используется **ограниченно и редко**
- [x] Auction не гарантирует конвертацию MC → GMC
- [x] Auction не создаёт стратегии фарма или накопления ради выгоды

---

## 4. MC Lifecycle & Safety

- [x] У MC существует жизненный цикл
- [x] Предусмотрено угасание MC без штрафной логики
- [x] Отсутствует бесконечное накопление
- [x] Отсутствует инфляция смысла при росте системы
- [x] Нет «оптимальных стратегий» эксплуатации экономики
- [x] Поведение человека важнее математической оптимальности

## 5. AI Guardrails (обязательные ограничения)

- [x] AI **не инициирует** экономические события
- [x] AI получает **только агрегированные данные**
- [x] Все рекомендации AI помечены как **non-binding**
- [x] Финальный арбитр всегда человек
