# Module 09 — Participation / Status / Ranks

## Назначение модуля

Данный модуль отвечает за **статусы и ранги участия человека в системе**.

Он фиксирует:
- профессионально-социальный статус (Status);
- форму участия в системе коинов (Rank);
- связанные с ними права, доступы и governance-эффекты.

Модуль **НЕ является частью MatrixCoin Economy** и **не влияет на расчёт MC**.

---

## Ключевые принципы

- Status = признание и роль человека.
- Rank = форма участия в системе коинов.
- Экономика (MC / GMC) остаётся изолированной и чистой.

Любое использование статуса или ранга для:
- умножения MC,
- прогнозирования дохода,
- KPI или мотивационного давления,

считается архитектурным нарушением.
